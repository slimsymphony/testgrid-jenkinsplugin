MANDATORY
This option has support for parameters. Default value is ${TESTGRID_ADDRESS}.
Format should be a json format object like:
[
{'id':'BROKER_TASK','uri':'nio://10.220.120.10:61616','mq':'ActiveMQ'},
{'id':'BROKER_STATUS','uri':'tcp://10.220.120.10:61617','mq':'ActiveMQ'},
{'id':'BROKER_FT','uri':'nio://10.220.120.10:61616','mq':'ActiveMQ'},
{'id':'BROKER_NOTIFICATION','uri':'tcp://10.220.120.10:61617','mq':'ActiveMQ'}
]

Or a simple URL like:
tcp://10.220.120.10:61616

NOTE: TESTGRID needs to be running in that address in order to work!
